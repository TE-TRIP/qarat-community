# QARAT コミュニティ - データ編集ガイド

このガイドは、メンバー情報とイベント情報を編集する際に必要なデータ構造を示しています。

---

## 📋 メンバー編集フォーム

メンバー情報を追加・編集する場合は、以下の情報を入力してください。

### メンバー情報テンプレート

```
【メンバーID】
yamane

【名前】
山根

【肩書き/職業】
カフェ・コミュニティマネージャ

【自己紹介/プロフィール】
コーヒーの概念が変わるようなコーヒー。QARATコミュニティマネージャ

【プロフィール画像URL】
https://raw.githubusercontent.com/TE-TRIP/qarat-community/main/assets/images/member-yamane.jpg

【専門分野（カンマ区切り）】
カフェ運営, コーヒー, コミュニティ運営

【提供サービス】
- サービス1名: カフェの運営相談
  サービス1URL: https://instagram.com/cafe.coffee.0_0
  
- サービス2名: コーヒー講座
  サービス2URL: https://instagram.com/cafe.coffee.0_0
  
- サービス3名: コミュニティ構築支援
  サービス3URL: https://instagram.com/cafe.coffee.0_0

【SNS/ウェブサイト】
Instagram: https://instagram.com/cafe.coffee.0_0
Twitter: （なければ空白）
Website: （なければ空白）
```

### メンバー情報の詳細説明

| フィールド | 説明 | 例 |
|-----------|------|-----|
| **メンバーID** | ユニークな識別子（英数字とハイフンのみ） | `yamane`, `member-2` |
| **名前** | メンバーの名前 | 山根 |
| **肩書き/職業** | メンバーの職業や役職 | フィットネスコーチ |
| **自己紹介** | プロフィール欄に表示される紹介文（100字程度） | パーソナルトレーニング指導15年。初心者から上級者まで対応。 |
| **プロフィール画像URL** | メンバーの顔写真URL（400×400px推奨） | https://... |
| **専門分野** | 得意分野（3〜5個、カンマ区切り） | 筋トレ, 栄養指導, ダイエット |
| **提供サービス** | メンバーが提供するサービス（2〜3個） | パーソナルトレーニング, 栄養相談 |
| **サービスURL** | サービスの詳細ページやSNS | https://instagram.com/... |
| **SNS/ウェブサイト** | Instagram, Twitter, Website（任意） | 少なくとも1つは入力推奨 |

## 🎉 イベント編集フォーム

イベント情報を追加・編集する場合は、以下の情報を入力してください。

```
【イベントID】
secret-cafe

【イベント名（日本語）】
シークレットカフェ

Secret Café

【簡潔な説明（1行）】
非公開の特別空間で、上質なひとときを。厳選されたメンバーだけが集う、特別なカフェイベントです。

【詳しい説明（3〜4行）】
QARATが厳選した非公開のカフェ・レストランを会場に、社会人同士が自然体で交流できる特別なイベントです。美味しいドリンクや軽食を楽しみながら、同じ価値観を持つ仲間と出会えます。会場は毎回異なり、その都度サプライズな空間をご用意しています。

【イベント画像URL】
https://raw.githubusercontent.com/TE-TRIP/qarat-community/main/assets/images/20250817-034A8959.jpg

【カテゴリ】
交流

【ハイライト（特徴）】
- 非公開の厳選会場
- 少人数制で深い交流
- ドリンク・軽食付き
- 毎回異なるテーマ

【開催スケジュール】
月1〜2回開催（土日祝日）

【定員】
10〜20名

【参加費】
参加費は各回ご案内

【申し込みフォームURL】
https://forms.google.com

【背景グラデーション色】
from-amber-50 to-orange-50
```

### イベント情報の詳細説明

| フィールド | 説明 | 例 |
|-----------|------|-----|
| **イベントID** | ユニークな識別子（英数字とハイフンのみ） | `secret-cafe`, `gym`, `run` |
| **イベント名（日本語）** | イベントの日本語名 | シークレットカフェ |
| **イベント名（英語）** | イベントの英語名（サブタイトル） | Secret Café |
| **簡潔な説明** | トップページに表示される1行説明（50字程度） | 非公開の特別空間で、上質なひとときを。 |
| **詳しい説明** | イベント詳細ページに表示される説明（200字程度） | QARATが厳選した非公開のカフェ・レストランを... |
| **イベント画像URL** | イベントのメイン画像URL（1200×800px推奨） | https://... |
| **カテゴリ** | イベントの分類 | `交流`, `フィットネス`, `ビューティー`, `クリエイティブ` |
| **ハイライト** | イベントの特徴（4個） | 非公開の厳選会場, 少人数制で深い交流, ... |
| **開催スケジュール** | 開催頻度と時間帯 | 月1〜2回開催（土日祝日） |
| **定員** | 参加人数の目安 | 10〜20名 |
| **参加費** | 費用情報 | 参加費は各回ご案内 |
| **申し込みフォームURL** | Google Forms等の申し込みリンク | https://forms.google.com |
| **背景グラデーション色** | Tailwind CSS のグラデーション色 | `from-amber-50 to-orange-50` |

### カテゴリの選択肢

| カテゴリ | 説明 | 推奨グラデーション |
|---------|------|------------------|
| **交流** | 交流・ネットワーキング系 | `from-amber-50 to-orange-50` |
| **フィットネス** | 運動・健康系 | `from-green-50 to-emerald-50` |
| **ビューティー** | 美容・ウェルネス系 | `from-pink-50 to-rose-50` |
| **クリエイティブ** | 創作・アート系 | `from-purple-50 to-indigo-50` |

## 🎨 背景グラデーション色の選択肢

## 📝 編集手順

### メンバーを追加・編集する場合

1. 上記のメンバー情報テンプレートに情報を入力
2. `client/src/lib/data.ts` の `MEMBERS` 配列に追加または編集
3. 以下のコマンドでビルド確認：
   ```bash
   pnpm run build
   ```
4. 変更をコミットして GitHub にプッシュ

### イベントを追加・編集する場合

1. 上記のイベント情報テンプレートに情報を入力
2. `client/src/lib/data.ts` の `EVENTS` 配列に追加または編集
3. 以下のコマンドでビルド確認：
   ```bash
   pnpm run build
   ```
4. 変更をコミットして GitHub にプッシュ

## 💡 入力時の注意点

### 画像URL について

- **推奨形式**: GitHub リポジトリの raw URL
  ```
  https://raw.githubusercontent.com/TE-TRIP/qarat-community/main/assets/images/image-name.jpg
  ```
- **サイズ目安**:
  - メンバー画像: 400×400px
  - イベント画像: 1200×800px
- 画像は `/assets/images/` ディレクトリに保存してください

### URL について

- SNS リンクは完全な URL を入力（`https://` から始まる）
- Google Forms の URL は共有可能な形式を使用

### テキスト入力について

- 改行は避け、1行で入力してください
- 特殊文字（`"`, `'`, `\`）は避けてください
- 絵文字は使用しないでください

## ❓ よくある質問

**Q: 新しいメンバーを追加するには？**
- メンバー情報テンプレートに入力して、`MEMBERS` 配列に追加してください。

**Q: イベントを削除するには？**
- `EVENTS` 配列からそのイベントを削除してください。

**Q: 画像を変更するには？**
- 新しい画像を `/assets/images/` にアップロードして、URL を更新してください。

**Q: 変更後、どのくらいで反映されますか？**
- GitHub にプッシュ後、GitHub Actions が自動的にビルド・デプロイします。通常 2〜5 分で反映されます。
